🔔 Уведомления на iPhone

  1. Откройте приложение через иконку на главном экране (не через Safari)
  2. Нажмите "Разрешить уведомления" выше
  3. В появившемся запросе нажмите "Разрешить"
  4. Перейдите в Настройки → Уведомления → Фичат
  5. Включите "Разрешить уведомления" и выберите нужные опции

📱 Инструкция для установки:

  1. Нажмите "Поделиться" (квадрат со стрелкой вверх)
  2. Выберите "На экран «Домой»"
  3. Добавьте приложение на главный экран
  4. Откройте приложение с главного экрана для получения уведомлений
